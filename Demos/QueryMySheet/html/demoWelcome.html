What this demo does:
Shows you how to query from a sheet directly into memory and place the results back into a new sheet.

Advanced Services Enabled:
Drive API v2 - for DevFlow
Google Sheets API v4 - for demo

Instructions:
Click Setup to create an input sheet (click again to reset), click Run to generate output using technique, click on Clean to remove all sheets besides Welcome, and click on X to close this demo.
Use the menu Add-ons>DevFlow Demos for Add-on Makers>Restart workflow to explore other demos.
